# Wedding Website Images Guide

This folder contains all images for your wedding website. Add your photos here following the naming conventions and dimensions listed below.

## üì∏ Image Placement Guide

### 1. Hero Background Image
**File name:** `hero-background.jpg`
**Recommended dimensions:** 1920 x 1080px (landscape)
**Purpose:** Background image for the top hero section with your names and date
**Notes:**
- Should be a beautiful but subtle image (soft focus works well)
- Consider a scenic Scottish landscape or a venue photo
- The image will have semi-transparent overlay with your names on top
- Will be cropped to fit screen, so keep important content centered

---

### 2. Couple Photo
**File name:** `couple-photo.jpg`
**Recommended dimensions:** 1200 x 800px or 800 x 600px (landscape)
**Purpose:** Featured photo in the "Our Story" section
**Notes:**
- A nice portrait of you two together
- Will be displayed centered with elegant shadow
- Best to use a high-quality, well-lit photo

---

### 3. Photo Gallery
**File names:** `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg`, `gallery-4.jpg` (add more as needed)
**Recommended dimensions:** 600 x 600px (square)
**Purpose:** Photo grid in the "Our Story" section
**Notes:**
- Square images work best for the grid layout
- Can include engagement photos, travel photos, or candid moments
- Hover effect adds nice interaction
- Grid automatically adjusts for different screen sizes

---

### 4. Venue Detail Card Images

#### Ceremony Venue
**File name:** `ceremony-venue.jpg`
**Recommended dimensions:** 600 x 400px (landscape/rectangular)
**Purpose:** Photo of ceremony location at Taypark House

#### Reception Venue
**File name:** `reception-venue.jpg`
**Recommended dimensions:** 600 x 400px (landscape/rectangular)
**Purpose:** Photo of the gardens or reception area

#### Dining Room
**File name:** `dining-room.jpg`
**Recommended dimensions:** 600 x 400px (landscape/rectangular)
**Purpose:** Photo of the dining area or table setup

**Notes:**
- These images appear at the top of each detail card
- Professional venue photos work great here
- Images should show the space in its best light

---

## üé® Image Optimization Tips

1. **File Format:** Use JPEG (.jpg) for photos - better compression than PNG
2. **File Size:** Keep images under 500KB each for faster loading
   - Use tools like TinyJPG.com or Squoosh.app to compress
3. **Quality:** Balance quality vs file size (80-85% quality is usually perfect)
4. **Aspect Ratios:** Stick to recommended dimensions for best results
5. **Lighting:** Use bright, well-lit photos when possible

## üìù How to Add Images

1. Save your photos with the exact file names listed above
2. Place them in this `images/` folder
3. The website will automatically display them
4. If you don't want to use certain images, you can:
   - Remove the image reference from `content.js`
   - Or simply don't add that image file (the site will gracefully hide missing images)

## üîÑ Updating Images

To change an image:
1. Replace the existing file with a new one using the same filename
2. Or update the filename in `content.js` to point to a different file

## ‚ú® Optional Images

All images are optional! If you don't add a specific image:
- Hero will show the elegant gradient background instead
- Detail cards will show just the icons
- Story section will show just text
- The site will still look beautiful
